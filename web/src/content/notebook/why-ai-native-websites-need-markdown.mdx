Human readers need hierarchy, taste, and restraint. Agents need dense context, canonical links, and facts that can survive extraction.

That means an AI-native website should have at least two surfaces:

- a human page with strong presentation
- a Markdown report with complete context and explicit relationships

The report is not hidden because it is secret. It is hidden because it is not the best way for a human to read.
